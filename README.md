# 📱 Smart Attendance Application

An Android-based Attendance Management System built using Kotlin and Jetpack Compose.

## 🚀 Features
- 🔐 Face Scan before Check-in
- 🆔 Class ID based student joining
- 👨‍🏫 Admin dashboard
- 📊 Present & Absent tracking
- ☁️ Firebase Backend Integration
- 🔑 Secure Login System

## 🛠️ Tech Stack
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- MVVM Architecture
